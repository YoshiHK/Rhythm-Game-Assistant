# Curator Labeling Guidelines

This document defines how curators should apply gold labels.

Principles:
- Judge correctness, not style
- Reference Phase 4 provenance when labeling
- Avoid speculative interpretation

Label types:
- tip_effectiveness: Did the tip help address the issue?
- severity_accuracy: Was the severity appropriate?
- element_relevance: Was the chosen element correct?
- recommendation_quality: Was the song recommendation appropriate?

Disagreements should be logged, not forced.
