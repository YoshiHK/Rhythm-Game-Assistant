# Phase 5 – Practice Integration

This folder defines the **Practice Integration & In‑Session Hints** subsystem for Phase 5 (Productionization).

Purpose:
- Bridge post‑hoc tips into guided practice experiences
- Enable opt‑in, assistive in‑session hints
- Collect practice telemetry for offline learning

Non‑negotiable rules:
- Downstream‑only (must not affect Phase 1–4.5 behavior)
- Assistive, not prescriptive or intrusive
- Fully opt‑in and user‑controllable
- No gameplay logic modification
