
# MVP vs FULL Taxonomy (Aligned)

## MVP
Minimal stable set for launch:
- EXEC_FAILURE
- PARTIAL_OUTPUT
- SELECTOR_FALLBACK_USED
- PERS_FALLBACK_LEGACY
- CAPABILITY_MISCLASSIFIED
- LOCALE_FALLBACK_USED
- PAYLOAD_EXTRACTION_FAIL
- RATIONALE_DEFAULT_USED

## FULL
Extends MVP to:
- Semantic / Feature layer
- Control & fallback systems
- API / orchestration layers
- Observability & learning layers

## Migration Rule
MVP reasons expand into FULL subcategories later.
